Definition at line 197 of file ApprecieModelBase.php.

decryptFields ( )

Decrypts fields indicated as encrypted, see setEncryptedFields(). Uses the EncryptionProvider provided by getEncryptionProvider() and the key provided by getEncryptionKey() Sensible defaults are provided.

This method is called implicitly before validation and after fetch operations

Note that after an update or create or validation action the object will be left in an unencrypted state, it will not be returned to its former state. An encrypted object will not double encrypt or double decrypt, this method has no effect on an encrypted object. Check the current encryption state by calling getIsDecrypted()
